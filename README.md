# deno-lisp

[Make a Lisp](https://github.com/kanaka/mal) implemented by TypeScript (Deno).  

## dev env

```bash
$ deno --version
deno 1.25.0 (release, x86_64-unknown-linux-gnu)
v8 10.6.194.5
typescript 4.7.4
```

## how to use

- run interpreter

```bash
./build.sh run
```

- run mal file

```bash
./build.sh <path/to/mal-file>
```

- test

```bash
./build.sh test
```
