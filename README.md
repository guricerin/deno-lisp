# deno-lisp

[Make a Lisp](https://github.com/kanaka/mal) implemented by TypeScript (Deno).  

## how to use

- run interpreter

```bash
./build.sh run
```

- run mal file

```bash
./build.sh <path/to/mal-file>
```

- test

```bash
./build.sh test
```
